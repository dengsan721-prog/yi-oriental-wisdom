# 「艺」命理软件 Demo 改版设计规格

日期：2026-07-17

## 1. 改版目标

产品回到清晰的命理使用逻辑：用户先建立出生坐标，再查看命盘、命理解读、大运流年和关系合盘。删除人物思想包装、专家谱系、长篇人生哲理和过量品牌说明。

页面信息比例控制为约 70% 命盘图形与数据、30% 简短解读。

## 2. 品牌启动页

页面只保留三个元素：

- 单字品牌「艺」
- 一句话「看见命局，读懂时运。」
- 主按钮「开始排盘」

不显示「东方命理全景推演」、人物背书、专家介绍或体系说明。视觉继续使用深海蓝黑、温润金和克制的五行时间环。

## 3. 核心流程

`品牌启动 → 生辰录入 → 命盘生成 → 命理总览 → 六大命理结果 → 大运流年 → 关系合盘 → 传统技法 → 趋吉避凶建议`

启动页点击「开始排盘」后进入独立录入页面，不在首屏堆叠功能介绍。

## 4. 生辰录入与快速年份选择

生辰采用独立年月日时选择器，不使用需要连续翻月的系统日期框。

- 年份范围为 1900 年至当前年份。
- 年份支持直接输入、十年分组跳转、上下滚动和最近选择。
- 月份使用 12 个月网格。
- 日期根据月份和闰年自动生成，非法日期不可选择。
- 时间支持具体时刻和十二时辰两种选择方式。
- 提供「大约年份」和「时辰不详」。
- 手机端使用底部滚轮或全屏选择面板；电脑端使用弹出面板。
- 完成选择后同步显示公历日期、农历日期、生肖和对应时辰，供用户核对。
- 时辰不详时降低结果置信度，只展示相对稳定的命理判断。

## 5. 命盘生成

生成过程依次点亮：

`四柱 → 五行 → 十神 → 格局 → 喜忌 → 大运`

动效服务于排盘过程，不加入人生哲理或专家语录。用户选择减少动态效果时，改为快速文本进度。

## 6. 命理总览

结果首屏必须直接呈现：

- 四柱八字
- 日主
- 五行旺衰图
- 十神结构
- 格局
- 喜用神与忌神
- 当前大运
- 一句总断

默认使用「创新叫法｜专业叫法」双层显示，同时保留通俗、双层、专业三个模式。

## 7. 六大命理结果

六张图形化卡片分别为：

1. 性格命势
2. 事业
3. 财运
4. 婚姻感情
5. 子女家庭
6. 健康节律

每张卡默认只展示核心判断、强弱或趋势图、典型场景和一条趋吉避凶建议。专业依据通过点击展开，不在页面默认铺开长文。

## 8. 大运流年

使用横向时间轴展示十年大运和年度节点。颜色、曲线高度和阶段标签表示变化；点击某个阶段才展开机会、风险、专业依据和建议。

不使用绝对吉凶、灾祸恐吓或确定性事件承诺。

## 9. 关系合盘

提供伴侣、子女、商业伙伴和朋友四种入口。合盘显示：

- 五行互补
- 十神关系
- 刑冲合害
- 沟通节奏
- 资源互补
- 相处建议

不输出单一「合／不合」分数，不使用克夫、克妻、克子等归罪表达。

## 10. 传统技法

面相、面痣、称骨、生肖和星座放在辅助工具区，不占据主报告核心位置。辅助结果明确标注不覆盖四柱主结论。照片流程继续采用本机预览模拟分析，并明确不上传、不保存。

## 11. 删除内容

从产品界面、GitHub Pages 和内容数据中删除：

- 南怀瑾相关内容
- 李叔同／弘一法师相关内容
- 倪海厦相关内容
- 开放式专家智慧谱系
- 历史人物镜鉴
- 长篇人生哲理和品牌理念段落
- 「东方命理全景推演」文案

## 12. 验收标准

- 打开首页只看到「艺」、一句文案和「开始排盘」。
- 用户在手机端可以快速选择出生年份，无需逐月翻页。
- 完成资料录入后首先看到专业命盘数据，而不是长篇说明。
- 页面主要视觉由命盘、刻度、强弱图、十神图和时间轴组成。
- 人物内容和专家谱系检索结果为零。
- 基础排盘测试继续通过。
- 手机与桌面无横向溢出，主要交互目标不小于 44px。
- GitHub Pages 与正式公网地址均发布新版。
